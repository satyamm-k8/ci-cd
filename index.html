🚀 Cloud-Based Application Hosting Architecture

Hello Bunny CI/CD Project 🔥

This project demonstrates a scalable cloud-based application hosting system using AWS, Docker, Jenkins, and GitHub.

🔧 Technologies Used:

AWS EC2 | Docker | Jenkins | GitHub

⚙️ How It Works:

The source code is stored in GitHub. Jenkins automatically pulls the latest code, builds a Docker image, and deploys the application on AWS EC2.

📈 Features:

✔ Continuous Integration & Deployment (CI/CD)
✔ Automated Build & Deployment
✔ Scalable Cloud Infrastructure
✔ Cost Efficient & High Availability

🎯 Conclusion:

This project shows how cloud computing and DevOps tools can be combined to build a reliable, scalable, and automated application hosting system.
